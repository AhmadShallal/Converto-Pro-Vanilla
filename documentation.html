Installation Guide

Follow these steps to get your ConvertPro template up and running in minutes:

Step 1: Download the package

After purchasing, download the ConvertPro package from your account dashboard or the email we sent you.

convertpro-v1.2.3.zip

Step 2: Extract the files

Extract the ZIP file to your preferred location. You'll see the following structure:

convertpro/
├── assets/
│   ├── css/
│   ├── js/
│   ├── images/
│   └── webfonts/
├── documentation/
└── index.html

Step 3: Set up your web server

Upload the files to your web server using FTP or your hosting provider's file manager.

For local development, you can use tools like XAMPP, MAMP, or a simple Node.js server.

Step 4: Customize your template

Open the files in your preferred code editor and start customizing the content, colors, and images to match your brand.

Customization Options

ConvertPro is designed to be easily customizable, even if you have minimal coding experience.

Changing Colors

To change the color scheme, edit the CSS variables in the assets/css/style.css file:

:root {
    /* Primary gradient colors */
    --primary-gradient: linear-gradient(135deg, #4353ff, #a466ff);
    --secondary-gradient: linear-gradient(135deg, #00d0ff, #4353ff);
    --accent-gradient: linear-gradient(135deg, #ff5e62, #ff9966);
    
    /* Solid colors */
    --primary: #4353ff;
    --primary-dark: #3242ee;
    --primary-light: #a466ff;
    --secondary: #00d0ff;
    --tertiary: #ff5e62;
    /* ... more color variables ... */
}

Typography

ConvertPro uses Google Fonts for typography. To change the font, update the link in the head of your HTML file and the font-family in the CSS.

Content and Images

Replace the placeholder content and images with your own. Make sure to maintain proper image aspect ratios for best results.

For best performance, optimize your images before uploading them to your server.
